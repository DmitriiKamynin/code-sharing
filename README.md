# Code Sharing App - Monorepo

Real-time collaborative code editor and execution platform with room support.

## Technologies

- **Backend:** NestJS, TypeScript, Socket.IO, BullMQ, PostgreSQL, TypeORM
- **Frontend:** React, TypeScript, Monaco Editor, Socket.IO Client
- **Engine:** Node.js, BullMQ
- **Infrastructure:** Docker, Nginx, Redis
- **Deployment:** Docker Compose
